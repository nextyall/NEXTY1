# WhatsApp Bot

A feature-rich WhatsApp bot with menu, ping, JID and forwarding capabilities.

## 🚀 Quick Deploy

[![Deploy to Heroku](https://www.herokucdn.com/deploy/button.svg)](https://heroku.com/deploy?template=https://github.com/nextyall/NEXTY1)

## 📋 Features
- `.menu` - Show menu
- `.ping` - Speed test  
- `.jid` - Get chat ID
- `.forward` - Forward messages

## ⚙️ Setup
1. Click "Deploy to Heroku" button
2. Get Session ID from: https://flex-pairing2-4ea9250ad369.herokuapp.com/pair
3. Add Session ID in Heroku Config Vars
4. Your bot will be ready!
